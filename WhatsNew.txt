
	FORA Notify - История изменений версий


   Версия 1.0

   Релиз!
   + Теперь есть возможность оставить отзыв на soft.zone66.su прямо из программы (сообщения появляются только после проверки).
   + Косметические изменения.


   Версия 0.9.5

   + Теперь при нажатии ктопки "закрыть" окно сворачивается в трей.
   + Косметические изменения.
   - Исправлены ошибки.


   Версия 0.9.4.1

   + Частично переписан код отвечающий за всплывающие окна.
   + Оптимизирован код отвечающий за смену тарифного плана.
   + Косметические изменения.
   - Исправлены небольшие ошибки.


   Версия 0.9.4

   + Добавлена возможность смены тарифного плана.
   + Косметические изменения.
   + Всплывающие окна теперь не появляются в случае когда запущено любое полноэкранное приложение.
   + Добавлен структурный обработчик исключений.
   - Исправления в парсере.
   - Исправлена ошибка, при которой не появлялось уведомление о низком балансе в случае когда сумма на счёте была отрицательной.
   - Исправлено воспроизведение звука.
   - Исправлено положение контекстного меню (вызываемого по нажатию правой кнопки мыши на иконке программы в трее).


   Версия 0.9.3

   - Исправлен вылет программы, который проявлялся при ошибке подключения.


   Версия 0.9.2

   + Уведомление о низком балансе (настраивается).
   + Косметические изменения.


   Версия 0.9.1

   Бета-версия
   + Логины и пароли сохраняются теперь в шифрованном виде.
   - Исправлена ошибка, при которой неправильно обрабатывался пароль в случае когда логин был длиннее пароля.
